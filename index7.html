URL is another word for a web address

URL: Uniform Resource Locator

Fomar of an URL: scheme://prefix.domain:port/path/filename

  • scheme: it defines the type of internet service: https, http, ftp, sftp
  • prefix: domain prefix, by default prefix is wwww
  • domain: internet domain name like amazon.com
  • port: port number at the host. default http port if 80
  • path: path at the server (not specified it takes the rootdirectory of the site)
  • filename: name of the document or resource

HTML vs XHTML

  • XHTML: Extensible Hyper Text Markup Language
  • It is more stricter, more XML-based version of HTML
  • XHMTL is HMTL defined as an application
  • XHTML supported by all majore browsers

Important differences

  • DOCTYPE is mandatory
  • xmlns attribute s mandatory
  • html, head, title, body are mandatory
  • Elements must always be properly nested
  • Elements must always be closed
  • Elements mus always be in lowecase
  • Attribute names must also be in lowecase
  • Attribute value must always be quoted
  • Attribute minimization is forbidden
