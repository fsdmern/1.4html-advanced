HTML Forms

Purpose: To collect user input and most of the times the input is sent to server for processing and storing

id:The for attribute of te label tag should be equal to id attribute of the input

name: input field must have a name. if name attribute is ommitted the value of the input will not be sent for processing

action: defines the action to be performed when the form is submitted

target: _blank, _self, _parent, _top => targt attribute specifies where to display the response that is received after submitting the form

method: get,post,patch,delete => method specifies HTTP method to be ised when form data is submitted

autocomplete: specifies whether a form should have autocomplete on or off

novalidate: it specifies that form data should not be validated
